README
======

Este repositório, de nome AJava-PracticeJulio, destina-se a pôr em prática a teoria das aulas do modulo 10 - Integração Continua e Qualidade de Software.

O objectivo do exercicio proposto para resolução em casa, pelo nosso formador António Sousa, é o seguinte:

	- Criar um novo repositório no GIT;
	- Criar um projecto em Java (exemplo, o "Hello World!"), adicioná-lo ao novo repositório criado anteriormente;
	- Enviar para o nosso formador António Sousa, convite para que ele faça a revisão das alterações;
